設計／ルール
  • css設計はFLOCSSをベースとする
    • Foundation - reset/normalize/base（ベース部分）
    • Layout - header/main/sidebar/footer（レイアウト）
    • Object
      • Component - grid/button/form/media（再利用可能なパーツ）「__」は使わない
      • Module - grid/button/form/media（再利用可能なパーツの集合体）
      • Project - articles/ranking/promo（プロジェクト独自で再利用できないもの）
      • Utility - clearfix/display/margin（調整用）
  • 各ページのスタイルは、common.css + ページcss（frontpage.cssなど）の組み合わせとし、不要なスタイルを読み込まないようにする（表示スピード対策）
  • セレクターにスタイルを設定しない。スタイルは全てクラスに設定すること
  • img要素にはwidth/heightを記載する（表示スピード対策）
  • img要素、iframe要素には「loading="lazy"」を記載する（表示スピード対策）
  • img要素の「alt」
  • アイコンは使用頻度の高いものをSVGで予め入れてあります。色を変えたいときはエディタで開いてカラーコードを変更してください
